# 기능 구현 목록

## 입력
- [x] `,` 단위로 자동차 이름을 구분하여 입력 받기
- [x] 시도 횟수 입력 받기

## 출력
- [ ] 각 라운드 마다 자동차의 이름과 위치 출력
- [ ] 최종 우승자 출력


## 자동차
- [x] 이름을 가진 자동차 생성
    - [x] 이름은 5자 이하
    - [x] 이름은 공백이 아닌 문자로 구성
- [x] 전진 조건에 따라 위치 변경
    - [x] 0~9 사이의 랜덤 값 생성
    - [x] 생성된 값이 4 이상일 경우 전진

## 게임
- [x] 참여한 자동차의 이름 중복 확인
- [ ] 시도 횟수 만큼 게임 진행
- [ ] 각 라운드 마다 자동차의 위치 변경
- [ ] 모든 라운드 종료 후 최종 우승자 판단
